Expand description

Memo

Brief

This struct defines a memo that is stored for each bag of the tree decomposition. They’re used for backtracking an optimal solution after the solver has finished its work.

pub fn intersect(&mut self, other: &TapsMemo)

Intersect two memos
Brief

Keeps only those entries in the memo that have at least one corresponding entry in another memo. Also modifies the stored values of the objective function accordingly.

Arguments
  • other : &TapsMemo - a memo of another bag of the tree decomposition.
Returns

None.
