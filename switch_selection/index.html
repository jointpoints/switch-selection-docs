Switch selection tool

Welcome

Welcome to the documentation of the switch selection tool! The switch selection tool was developed to demonstrate the possibilities of tree-decomposition based dynamic programming algorithms for problems arising in power systems, or more specifically, distribution grids. The program was written as a part of our submission to PSCC 2024 and is primarily intended for scientific research.

Purpose of documentation

This documentation is not a manual on how to use or compile the program. For that, see the README.md file in our repository.

Instead, this documentation provides an insight into the internal structure of our code, the relationships between different structs, modules, traits, etc., and explains what each element does. If you want to know more about the algorithm of a particular function, you can look at its source code. The source code contains comments explaining what is done and why.
